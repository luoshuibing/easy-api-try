# 变更日志

## 0.1.0
- 更名：扩展名称改为`easy-api-try`，显示名为`Easy API Try`
- 新增命令：`扫描Spring Boot接口并打开调试器`
- 扫描`@RestController/@Controller`类与多种`*Mapping`注解
- 支持类级`@RequestMapping`前缀与无路径`@PostMapping()`
- Webview就绪握手，支持刷新重新扫描接口
- 接口选中高亮、路径变量输入自动生成
- 请求头表格增删，内置基础默认头
- 一键格式化请求体JSON
- 扩展侧HTTP/HTTPS请求，输出面板与Webview Console日志
